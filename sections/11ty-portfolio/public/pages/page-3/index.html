New York City in Data

↖ Link to Website
Core 2 Interaction Studio
Apr–May 2023
HTML, CSS, JS
This website visualizes the Popular Baby Names dataset from NYC Open Data, showcasing the frequency and popularity of baby names in New York City. The website offers a random name generator to explore popular names using data from 2011 to 2019. It provides an overview and trends in baby names during that period, allowing users to select specific years and ranking options.
